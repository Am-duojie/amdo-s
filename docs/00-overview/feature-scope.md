# 功能范围清单（毕业设计版）

## 用户端
- 账号与认证：注册/登录、个人信息、头像上传
- 商品：分类浏览、发布/编辑/上下架、图片上传、我的商品/我的售卖
- 交易订单：下单、状态更新（待付款/已付款/已发货/已完成/已取消）
- 收藏：收藏/取消收藏、收藏列表
- 私信：会话列表、消息发送/已读、撤回、图片消息（如实现）
- 地址：地址增删改查、下单使用
- 钱包：余额查询、提现（如实现）
- 回收：机型目录、估价问卷、提交回收订单、订单列表与详情

## 官方验（用户端展示）
- 官方验商品列表与详情
- 官方验下单与订单状态（如实现）
- 官方验收藏（如实现）

## 管理端
- 登录/登出/刷新 token/修改密码
- 权限点与菜单
- 仪表盘与统计
- 回收订单管理：筛选、详情、物流、定价、打款、发布为官方验、批量更新
- 官方验设备库存：列表/详情、从回收订单生成设备、上架/下架动作
- 审核队列：决策、日志
- 支付订单与结算、结算汇总
- 基础数据：分类、商品、用户、消息、地址（按实现）

## 非功能（建议写入论文）
- 安全：JWT、回调验签、敏感配置不入库
- 可靠性：支付回调幂等、状态机约束、日志留痕
- 可维护性：模块化、文档与变更流程（见 maintenance）
