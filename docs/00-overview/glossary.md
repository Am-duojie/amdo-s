# 术语表

- 二手商品（Product）：用户发布的普通商品
- 官方验商品（VerifiedProduct）：通过回收质检或运营发布的官方验商品
- 回收订单（RecycleOrder）：用户发起的回收流程订单（估价→确认→寄送→检测→打款/完成）
- 检测报告（AdminInspectionReport）：管理端对回收订单的质检记录
- 打款状态（payment_status）：回收订单打款进度（与流程状态 status 分离）
- 订单（Order/VerifiedOrder）：用户购买普通商品/官方验商品的交易订单
- 管理端账号（AdminUser）：独立于前台用户的后台登录账户
- 角色权限（AdminRole.permissions）：以 JSON 列表保存的权限点集合
