# 变更单：回收主页UI优化

**变更ID**: 2024-12-13-recycle-ui-optimization

## 变更概述

对回收主页进行全面的UI优化，创建统一的设计系统和组件库，提升用户体验和视觉一致性。

## 主要变更内容

### 1. 设计系统建立
- 统一颜色系统（文字、背景、边框）
- 标准化间距系统（4px基准）
- 规范化圆角系统（4px-24px）
- 完善阴影系统（5个层级）
- 定义过渡动画规范

### 2. 组件库创建
- **PageContainer**: 统一页面布局容器
- **BaseCard**: 通用卡片组件，支持多种配置

### 3. 回收主页优化
- 采用现代化网格布局（左侧引导 + 右侧选择器）
- 优化左侧引导卡片，增加视觉吸引力
- 改进右侧机型选择器，提升交互体验
- 增加丰富的hover效果和过渡动画
- 优化响应式布局，改善移动端体验

### 4. 交互体验提升
- 卡片hover效果：上移 + 阴影加深
- 按钮hover效果：轻微上移 + 彩色阴影
- 列表项hover效果：右移 + 边框高亮
- 统一过渡动画：0.3s cubic-bezier缓动
