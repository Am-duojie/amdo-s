# 2025-12-19 统计分析：回收履约效率与异常

## 目的

在管理端统计页补充“漏斗 + 时效分布 + 异常占比”，用于答辩/演示的流程效率分析。

## 变更内容

- 后端统计接口新增回收履约效率数据：
  - 环节耗时分布（中位数/P90/P95）
  - 异常占比（取消率、价格异议率、打款失败率、质检后未确认）
- 新增超时占比（SLA）：
  - 创建→寄出 > 72 小时
  - 寄出→收货 > 48 小时
- 新增调价幅度分布：
  - 统计 |final - estimated| / estimated 的区间分布，用于解释估价准确度
  - 补充样本量与平均偏差提示
  - 导出 CSV 增加调价分布与 SLA 指标
- 新增 SLA 责任归因：
  - 用户寄出（创建→寄出）
  - 物流（寄出→收货）
  - 质检（收货→质检）
- 新增回收 KPI（完成率、取消率、均值单价）
- 新增易淘 KPI 与价格分布：
  - 完成率、取消率、均值单价
  - 价格区间分布（已付款订单）
- 前端统计页新增“回收履约效率与异常”模块。

## 验证方式

1. 打开管理端 `统计分析` 页面
2. 确认出现“回收履约效率与异常”模块
3. 随机切换日期范围，查看数值随数据变化
4. 检查“超时占比（SLA）”表格是否显示
5. 检查“调价幅度分布（估价偏差）”柱状图是否显示
6. 导出 CSV，确认包含调价分布与 SLA 字段
7. 检查 SLA 责任归因标签是否显示
